# 🗺️ 地图编辑器使用指南

## 访问地址
- 本地访问: http://localhost:3001/tools/map-editor
- 局域网HTTPS访问: https://192.168.110.169/tools/map-editor

## 核心功能

### 1. 素材自动检测
- 从下拉菜单选择素材图片
- 点击"🔄 加载素材"按钮
- 系统会自动使用BFS算法检测所有独立素材
- 检测到的素材会显示在左侧素材库中

### 2. 三种编辑工具

#### 👆 选择工具（默认）
- **放置素材**: 先在左侧选中一个素材，然后在画布上点击放置
- **移动素材**: 点击并拖动已放置的素材
- **缩放素材**: 选中素材后，拖动四个角的绿色手柄进行等比缩放
  - 左上角: 向左上缩放
  - 右上角: 向右上缩放
  - 左下角: 向左下缩放
  - 右下角: 向右下缩放（最常用）

#### ⬜ 碰撞框工具
- 先用选择工具选中一个素材
- 切换到碰撞框工具
- 在画布上拖动绘制碰撞区域（红色虚线框）
- 松开鼠标完成绘制
- 一个素材可以有一个碰撞框
- 可以点击"移除碰撞框"按钮删除碰撞框

#### 🗑️ 删除工具
- 点击任意素材即可删除

### 3. Y轴深度排序
- 所有素材会自动按照 `y + height` 进行排序
- 这样可以实现自然的前后遮挡效果
- 例如：玩家在建筑物后面时会被遮挡，在前面时会遮挡建筑物

### 4. 导出/导入地图
- **导出**: 点击"📋 导出JSON"，地图数据会显示在文本框中
- **导入**: 将JSON数据粘贴到文本框，点击"📥 导入JSON"

## 地图数据格式

```json
{
  "width": 960,
  "height": 720,
  "objects": [
    {
      "id": 1234567890,
      "spriteIndex": 0,
      "x": 100,
      "y": 200,
      "width": 96,
      "height": 144,
      "collision": {
        "x": 110,
        "y": 250,
        "width": 76,
        "height": 50
      }
    }
  ]
}
```

## 使用场景

### 场景1: 创建城市街道
1. 加载"城市建筑"素材
2. 选择建筑物素材
3. 在画布上放置多个建筑
4. 使用碰撞框工具为每个建筑的底部（门口区域）添加碰撞
5. 这样玩家可以从建筑后面走过，但不能穿过门口

### 场景2: 室内家具布局
1. 加载"家具动画"素材
2. 放置桌子、椅子等家具
3. 为每个家具添加碰撞框
4. Y轴排序会自动处理家具的前后关系

## 技术特性

- **智能检测**: BFS算法自动识别连通区域
- **跨域支持**: 支持HTTPS访问时从HTTP后端加载资源
- **等比缩放**: 保持素材原始宽高比
- **实时渲染**: 所有操作立即反映在画布上
- **深度排序**: 自动按Y坐标排序，无需手动调整图层
