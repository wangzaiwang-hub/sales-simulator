# 功能：自动滚动到最新消息

## 功能说明

现在聊天窗口会自动滚动到最新的消息，方便查看最近的对话。

## 实现方式

### 1. 添加滚动容器引用

```typescript
const chatScrollRef = useRef<HTMLDivElement>(null);
```

### 2. 添加自动滚动逻辑

```typescript
// 自动滚动到最新消息
useEffect(() => {
  if (chatScrollRef.current && selectedNpcId && chatMessages[selectedNpcId]?.length > 0) {
    // 使用setTimeout确保DOM已更新
    setTimeout(() => {
      if (chatScrollRef.current) {
        chatScrollRef.current.scrollTop = chatScrollRef.current.scrollHeight;
      }
    }, 100);
  }
}, [chatMessages, selectedNpcId]);
```

### 3. 绑定ref到滚动容器

```tsx
<div ref={chatScrollRef} className="h-full overflow-auto ...">
  {/* 聊天消息 */}
</div>
```

## 工作原理

### 触发时机

自动滚动会在以下情况触发：

1. **加载聊天历史后**
   - 打开聊天窗口
   - 从数据库加载历史记录
   - 自动滚动到最新消息

2. **发送新消息后**
   - 用户发送消息
   - NPC回复
   - 自动滚动到最新消息

3. **切换NPC后**
   - 关闭当前聊天
   - 打开另一个NPC的聊天
   - 自动滚动到最新消息

### 滚动行为

```javascript
// 滚动到底部
chatScrollRef.current.scrollTop = chatScrollRef.current.scrollHeight;
```

- `scrollTop`：当前滚动位置
- `scrollHeight`：内容总高度
- 设置`scrollTop = scrollHeight`即滚动到底部

### 延迟处理

```javascript
setTimeout(() => {
  // 滚动逻辑
}, 100);
```

使用100ms延迟确保：
- DOM已完全更新
- 新消息已渲染
- 高度已计算完成

## 技术细节

### 依赖项

```typescript

  // ...

```

- `chatMessages`：聊天消息变化时触发
- `selectedNpcId`：切换NPC时触发

### 条件检查

```typescript
if (chatScrollRef.current && selectedNpcId && chatMessages[selectedNpcId]?.length > 0) {
  // 滚动逻辑
}
```

确保：
1. 滚动容器存在
2. 有选中的NPC
3. 有聊天消息

### 平滑滚动（可选）

如果想要平滑滚动效果，可以使用：

```typescript
chatScrollRef.current.scrollTo({
  top: chatScrollRef.current.scrollHeight,
  behavior: 'smooth'
});
```

但目前使用的是瞬间滚动，因为：
- 更快
- 更符合聊天应用的习惯
- 避免用户等待

## 可能的改进

### 改进1：智能滚动

只在用户已经在底部时才自动滚动：

```typescript
const isNearBottom = 
  chatScrollRef.current.scrollHeight - chatScrollRef.current.scrollTop - chatScrollRef.current.clientHeight < 100;

if (isNearBottom) {
  // 滚动到底部
}
```

### 改进2：滚动指示器

当有新消息但用户不在底部时，显示提示：

```tsx
{!isAtBottom && hasNewMessages && (
  <button onClick={scrollToBottom}>
    有新消息 ↓
  </button>
)}
```

### 改进3：记住滚动位置

切换NPC后返回时，恢复之前的滚动位置：

```typescript
const scrollPositions = useRef<Record<string, number>>({});

// 保存位置
scrollPositions.current[selectedNpcId] = chatScrollRef.current.scrollTop;

// 恢复位置
chatScrollRef.current.scrollTop = scrollPositions.current[selectedNpcId] || scrollHeight;
```
