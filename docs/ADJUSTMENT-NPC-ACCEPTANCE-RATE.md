# NPC接受率调整说明

## 问题

用户反馈：找别的NPC聊天，一直都是不想和我聊天。

## 原因分析

之前的设置过于严格，导致陌生人拒绝率太高：

### 旧设置的问题

1. **社恐门槛太低**：外向性 < 25 就有75%拒绝率
2. **内向拒绝率太高**：外向性 < 40 就有50%拒绝率
3. **神经质拒绝率高**：神经质 > 65 就有40%拒绝率
4. **开放性拒绝率高**：开放性 < 35 就有35%拒绝率
5. **最终接受率计算没有基础分**：导致性格一般的NPC接受率只有50%左右

### 实际效果

大部分NPC的性格特征都在40-60之间（正态分布），这导致：
- 外向性50的NPC，基础接受率只有 (50*0.6 + 50*0.4)/100 = 50%
- 再加上各种拒绝条件，实际接受率可能只有30-40%
- 用户体验很差，感觉NPC都不愿意聊天

## 调整方案

### 1. 降低陌生人拒绝率

**社恐门槛**：
- 旧：外向性 < 25，拒绝率75%
- 新：外向性 < 20，拒绝率60%
- 效果：只有极度内向的人才会频繁拒绝

**内向拒绝率**：
- 旧：外向性 < 40，拒绝率50%
- 新：外向性 < 30，拒绝率30%
- 效果：一般内向的人也会接受聊天

**神经质拒绝率**：
- 旧：神经质 > 65，拒绝率40%
- 新：神经质 > 75，拒绝率25%
- 效果：只有极度敏感的人才会警惕陌生人

**开放性拒绝率**：
- 旧：开放性 < 35，拒绝率35%
- 新：开放性 < 25，拒绝率20%
- 效果：只有极度保守的人才会拒绝新接触

### 2. 提高基础接受率

**旧公式**：
```typescript
acceptChance = (extraversion * 0.6 + agreeableness * 0.4) / 100
// 外向性50，宜人性50 → 接受率 = 50%
```

**新公式**：
```typescript
acceptChance = (extraversion * 0.5 + agreeableness * 0.3 + 30) / 100
// 外向性50，宜人性50 → 接受率 = 70%
// 加了30基础分，确保大部分NPC愿意聊天
```

**最高接受率限制**：
```typescript
finalChance = Math.min(0.95, acceptChance + bonuses)
// 最高95%，保留5%的随机性
```

### 3. 调整其他拒绝条件

**心情不好**：
- 旧：神经质 > 60，拒绝率60%
- 新：神经质 > 70，拒绝率40%

**社交能量低**：
- 旧：能量 < 20 且 外向性 < 50
- 新：能量 < 15 且 外向性 < 40

**宜人性低**：
- 旧：宜人性 < 35，拒绝率30%
- 新：宜人性 < 25，拒绝率20%

## 调整效果

### 不同性格的接受率对比

| 性格特征 | 旧接受率 | 新接受率 | 变化 |
|---------|---------|---------|------|
| 外向50，宜人50 | ~50% | ~70% | +20% |
| 外向60，宜人60 | ~60% | ~78% | +18% |
| 外向40，宜人40 | ~40% | ~62% | +22% |
| 外向30，宜人30 | ~30% | ~54% | +24% |
| 外向70，宜人70 | ~70% | ~86% | +16% |

### 极端性格的接受率

| 性格类型 | 旧接受率 | 新接受率 | 说明 |
|---------|---------|---------|------|
| 极度外向（90） | ~80% | ~95% | 几乎总是接受 |
| 极度内向（10） | ~10% | ~40% | 仍然较低，但不是完全拒绝 |
| 社恐（外向15） | ~5% | ~30% | 有机会聊天，但较难 |
| 友善（宜人80） | ~70% | ~85% | 很容易接受 |
| 不友善（宜人20） | ~30% | ~50% | 仍然有机会 |

## 设计理念

### 1. 保持真实性

- 极度内向的人仍然会拒绝陌生人（但不是100%）
- 性格开朗的人更容易接受聊天
- 心情不好、压力大时会拒绝

### 2. 提升可玩性

- 大部分NPC愿意和陌生人聊天（60-80%接受率）
- 玩家不会频繁碰壁
- 仍然保留一些拒绝的可能性，增加真实感

### 3. 关系成长

- 陌生人阶段：60-70%接受率
- 熟人阶段：70-80%接受率
- 朋友阶段：90%+接受率
- 好友阶段：几乎100%接受率

## 测试建议

### 测试场景1：普通NPC

1. 找一个外向性50左右的NPC
2. 尝试聊天5次
3. 预期：3-4次接受，1-2次拒绝

### 测试场景2：外向NPC

1. 找一个外向性70+的NPC
2. 尝试聊天5次
3. 预期：4-5次接受，0-1次拒绝

### 测试场景3：内向NPC

1. 找一个外向性30以下的NPC
2. 尝试聊天5次
3. 预期：2-3次接受，2-3次拒绝

### 测试场景4：关系成长

1. 找一个NPC，从陌生人开始
2. 多次聊天，提升好感度和熟悉度
3. 观察接受率是否提高
4. 预期：成为朋友后几乎不会拒绝

## 后续优化

### 可能的改进方向

1. **动态社交能量**：根据NPC的活动状态动态计算
2. **时间因素**：早上精力充沛，晚上容易疲惫
3. **场景因素**：在咖啡厅更容易聊天，在工作场所更容易拒绝
4. **群体效应**：周围有其他人时，外向的人更愿意聊天
5. **历史记录**：如果最近被拒绝过，下次接受率提高

### 数据收集

建议收集以下数据来优化参数：
- 不同性格NPC的实际接受率
- 用户的聊天尝试次数和成功率
- 用户反馈（是否觉得太容易/太难）
