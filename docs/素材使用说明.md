## 🎨 什么是Sprite Sheet？

**Sprite Sheet（精灵图集）** 是将多个动画帧或图像组合在一张大图中的技术。

### 为什么使用Sprite Sheet？
1. **减少文件数量** - 一张图包含所有帧，而不是几十个单独的文件
2. **提高性能** - 游戏引擎只需加载一张图
3. **方便管理** - 所有相关动画在一起

### 示例：
```
行走动画Sprite Sheet（8帧）：
[帧1][帧2][帧3][帧4][帧5][帧6][帧7][帧8]
```

游戏会按顺序显示这些帧，创造出行走的动画效果。

## 🗺️ 什么是Tileset？

**Tileset（瓦片集）** 是用于构建游戏地图的小方块集合。

### 如何使用Tileset？
1. 每个小方块（tile）通常是固定尺寸（如16x16、32x32、48x48）
2. 使用地图编辑器（如Tiled）将这些方块拼接成完整场景
3. 就像用乐高积木搭建场景一样！

## 💻 如何在代码中使用这些素材？

### 方法1：原生JavaScript + Canvas

```javascript
// 1. 加载sprite sheet
const spriteSheet = new Image();
spriteSheet.src = 'path/to/Character_Walk_Front-Sheet.png';

// 2. 定义帧信息
const frameWidth = 32;   // 每帧宽度
const frameHeight = 32;  // 每帧高度
const totalFrames = 8;   // 总帧数
let currentFrame = 0;

// 3. 绘制当前帧
function drawFrame() {
    ctx.drawImage(
        spriteSheet,
        currentFrame * frameWidth, 0,  // 从sprite sheet的哪里切割
        frameWidth, frameHeight,        // 切割尺寸
        x, y,                           // 在画布上的位置
        frameWidth, frameHeight         // 显示尺寸
    );
}

// 4. 动画循环
function animate() {
    currentFrame = (currentFrame + 1) % totalFrames;
    drawFrame();
    requestAnimationFrame(animate);
}
```

### 方法2：使用Phaser.js游戏框架

```javascript
// 1. 预加载sprite sheet
function preload() {
    this.load.spritesheet('player_walk', 
        'frontend/public/32x32 Customizable Character Pack/Walk/Character/Character_Walk_Front-Sheet.png',
        { frameWidth: 32, frameHeight: 32 }
    );
}

// 2. 创建动画
function create() {
    this.anims.create({
        key: 'walk',
        frames: this.anims.generateFrameNumbers('player_walk', { start: 0, end: 7 }),
        frameRate: 10,
        repeat: -1
    });
    
    // 3. 创建精灵并播放动画
    const player = this.add.sprite(400, 300, 'player_walk');
    player.play('walk');
}
```

### 方法3：使用Unity

1. 将sprite sheet导入Unity
2. 在Inspector中设置Sprite Mode为"Multiple"
3. 点击"Sprite Editor"切割sprite sheet
4. 使用Animation窗口创建动画
5. 将动画帧拖入时间轴

### 方法4：使用Godot

1. 导入sprite sheet到项目
2. 创建AnimatedSprite节点
3. 在SpriteFrames中添加动画
4. 设置每帧的区域（Region）
5. 调用`play("animation_name")`播放

## 💡 实用技巧

### 1. 图层叠加
你的32x32角色包可以叠加多个图层：
```javascript
// 按顺序绘制图层
drawLayer(characterBody);
drawLayer(clothing);
drawLayer(hair);
drawLayer(eyes);
```

### 2. 动画状态机
管理不同动画状态：
```javascript
const states = {
    IDLE: 'idle',
    WALK: 'walk',
    RUN: 'run',
    JUMP: 'jump'
};

let currentState = states.IDLE;

function changeState(newState) {
    if (currentState !== newState) {
        currentState = newState;
        // 切换到对应的sprite sheet
    }
}
```

### 3. 性能优化
- 使用sprite atlases（图集）减少绘制调用
- 只更新移动的对象
- 使用对象池避免频繁创建/销毁对象

## ❓ 常见问题

### Q: 为什么我的动画看起来很卡？
A: 调整帧率。通常8-12 FPS适合像素艺术动画。

### Q: 如何改变角色颜色？
A: 使用Canvas的`globalCompositeOperation`或在游戏引擎中使用着色器。

### Q: 可以商用这些素材吗？
A: 查看每个素材包中的License文件。你的32x32角色包是CC0许可（完全免费使用）。
