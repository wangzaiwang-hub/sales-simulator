# 🎨 瓦片地图编辑器 - 图层系统使用说明

## ✨ 新功能概述

现在编辑器支持多图层系统，让你可以创建更复杂、更丰富的地图！

## 📚 什么是图层？

图层就像透明的纸张叠在一起：
- **背景层**：放置地面、草地等基础瓦片
- **装饰层**：放置树木、建筑、家具等装饰物
- **前景层**：放置需要显示在最上面的元素

瓦片会按照图层顺序从下到上绘制，实现重叠效果。

## 🎯 图层功能

### 1. 图层列表
- 显示所有图层
- 当前编辑的图层会高亮显示（绿色背景）
- 非当前图层在画布上会稍微透明（60%透明度）

### 2. 切换图层
- **点击图层名称**：切换到该图层进行编辑
- 只有当前图层可以被编辑

### 3. 可见性控制
- **复选框** ☑️：控制图层是否显示
- 取消勾选可以隐藏图层，方便编辑其他图层
- 隐藏的图层不会在测试时显示

### 4. 锁定图层
- **锁图标** 🔒/🔓：点击切换锁定状态
- 锁定的图层无法编辑，防止误操作
- 🔒 = 已锁定，🔓 = 未锁定

### 5. 添加/删除图层
- **➕ 添加图层**：创建新图层，会提示输入图层名称
- **➖ 删除当前图层**：删除当前选中的图层（至少保留1个图层）

## 🎮 使用流程示例

### 创建一个城市场景：

1. **背景层**（默认已有）
   - 选择地面瓦片
   - 铺满整个地图作为基础

2. **添加"道路层"**
   - 点击"➕ 添加图层"
   - 输入名称"道路层"
   - 绘制道路

3. **添加"建筑层"**
   - 再次点击"➕ 添加图层"
   - 输入名称"建筑层"
   - 放置建筑物、树木等

4. **添加"装饰层"**
   - 添加最后一个图层
   - 放置路灯、标志等小装饰

## 🔄 变换功能

每个瓦片都支持独立的变换：
- **旋转**：0°, 90°, 180°, 270°（快捷键：R）
- **水平翻转**：镜像翻转（快捷键：H）
- **垂直翻转**：上下翻转（快捷键：V）

变换会应用到当前选中的瓦片，绘制时保持变换状态。

## 💾 导出/导入

### 导出
- 点击"导出JSON"按钮
- 数据会自动复制到剪贴板
- 包含所有图层、瓦片位置和变换信息

### 导入
1. 将导出的JSON数据粘贴到文本框
2. 点击"导入JSON"按钮
3. 地图会完整恢复，包括所有图层

## 🧪 测试地图

- 点击"测试地图"按钮
- 会在新窗口打开测试页面
- 显示所有可见图层的完整效果
- 可以控制角色在地图上移动

## ⌨️ 快捷键

- **B**：切换到画笔工具
- **E**：切换到橡皮工具
- **R**：旋转瓦片 90°
- **H**：水平翻转
- **V**：垂直翻转

## 💡 使用技巧

1. **分层思维**：将不同类型的元素放在不同图层，便于管理
2. **隐藏图层**：编辑某一层时，可以隐藏其他图层看得更清楚
3. **锁定保护**：完成的图层可以锁定，避免误修改
4. **多选瓦片**：在瓦片集上拖动鼠标可以选择多个瓦片，一次性绘制
5. **变换预览**：选中瓦片后，预览窗口会显示变换后的效果

## 🎨 瓦片共享

所有图层共享同一个瓦片集，这意味着：
- 瓦片只是素材，可以在地图任何位置重复使用
- 不同图层可以使用相同的瓦片
- 同一个瓦片可以应用不同的变换（旋转、翻转）
