# 功能：聊天窗口头部重设计

## 新设计说明

重新设计了聊天窗口的头部，使其更加直观和美观。

## 设计布局

```
┌─────────────────────────────────────────┐
│  QQ CHAT                        [关闭]  │
├─────────────────────────────────────────┤
│  👤        ━━━━━━━━━━━━━━━        👤   │
│  玩家      好感度: 25 ▓▓▓▓░░░░    NPC  │
│  头像      熟悉度: 30 ▓▓▓░░░░░    头像  │
│                                         │
│           junqianxi.hub                 │
│        行政官员 · 社交 · 探索           │
└─────────────────────────────────────────┘
```

## 功能特点

### 1. 双头像显示

#### 左侧：玩家头像
- 圆形头像
- 渐变背景（青色到蓝色）
- 显示用户名首字母
- 尺寸：56x56px

#### 右侧：NPC头像
- 圆形头像
- 渐变背景（粉色到紫色）
- 显示NPC名字首字母
- 尺寸：56x56px

### 2. 中间进度条区域

#### 正常状态：显示关系进度条

**好感度进度条**
- 范围：-100 到 +100
- 显示当前数值
- 渐变颜色：红色 → 黄色 → 绿色
- 动画过渡：500ms

**熟悉度进度条**
- 范围：0 到 100
- 显示当前数值
- 渐变颜色：蓝色 → 紫色
- 动画过渡：500ms

#### 加载状态：正在输入中

当AI正在回复时：
- 隐藏进度条
- 显示"对方正在输入中..."
- 三个跳动的小圆点动画
- 半透明白色背景

### 3. NPC信息

- 居中显示NPC名字
- 显示职业和兴趣（最多2个）
- 小字体，半透明

## 实现细节

### 获取最新关系数据

```typescript
const messages = chatMessages[selectedNpc.id] || [];
const lastNpcMessage = [...messages].reverse().find(m => m.role === 'npc' && m.relationship);
const affinity = lastNpcMessage?.relationship?.affinity ?? 0;
const familiarity = lastNpcMessage?.relationship?.familiarity ?? 0;
```

从聊天记录中获取最后一条NPC回复的关系数据。

### 好感度进度条计算

```typescript
// 好感度范围：-100 到 +100
// 转换为 0% 到 100%
const affinityPercent = ((affinity + 100) / 200) * 100;
```

### 熟悉度进度条计算

```typescript
// 熟悉度范围：0 到 100
// 直接使用百分比
const familiarityPercent = familiarity;
```

### 正在输入动画

```tsx
<div className="flex gap-1">
  <span className="w-2 h-2 bg-slate-800 rounded-full animate-bounce" 
        style={{ animationDelay: '0ms' }}></span>
  <span className="w-2 h-2 bg-slate-800 rounded-full animate-bounce" 
        style={{ animationDelay: '150ms' }}></span>
  <span className="w-2 h-2 bg-slate-800 rounded-full animate-bounce" 
        style={{ animationDelay: '300ms' }}></span>
</div>
```

三个小圆点，延迟跳动，形成波浪效果。

## 视觉效果

### 颜色方案

**玩家头像**
- 渐变：`from-cyan-400 to-blue-500`
- 代表玩家的冷静和理性

**NPC头像**
- 渐变：`from-pink-400 to-purple-500`
- 代表NPC的个性和魅力

**好感度进度条**
- 渐变：`from-red-400 via-yellow-400 to-green-400`
- 红色（负面）→ 黄色（中性）→ 绿色（正面）

**熟悉度进度条**
- 渐变：`from-blue-400 to-purple-400`
- 蓝色（陌生）→ 紫色（熟悉）

### 动画效果

1. **进度条过渡**
   - `transition-all duration-500`
   - 平滑的宽度变化

2. **跳动动画**
   - `animate-bounce`
   - 三个圆点依次跳动

3. **按钮悬停**
   - `hover:bg-white/60 transition`
   - 关闭按钮悬停变亮
