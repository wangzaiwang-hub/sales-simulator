# 🎮 开始局域网游戏

## 当前状态

✅ 前端服务运行中 (3000端口)
✅ 后端服务运行中 (3001端口)  
✅ 局域网IP: **192.168.110.169**
❌ nginx未安装

## 只需两步

### 第一步：安装nginx

```bash
brew install nginx
```

### 第二步：启动局域网服务

**推荐方案（80端口）**：
```bash
./start-lan.sh
```

访问地址：`http://192.168.110.169`
SecondMe回调：`http://192.168.110.169/auth/callback`

**或者使用8080端口（无需sudo）**：
```bash
./start-lan-8080.sh
```

访问地址：`http://192.168.110.169:8080`
SecondMe回调：`http://192.168.110.169:8080/auth/callback`

## 配置SecondMe

启动后，脚本会显示需要添加的回调地址。

去SecondMe平台添加对应的回调地址即可。

## 开始游戏

在局域网内任何设备的浏览器打开对应的访问地址。

## 检查环境

随时运行此命令检查环境状态：
```bash
./check-lan-ready.sh
```

## 停止服务

```bash
./stop-lan.sh
```
