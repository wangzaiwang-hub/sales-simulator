# 🎮 局域网多人游戏 - 快速开始

## 三步启动

### 步骤1：启动服务
```bash
./start-lan.sh
```

### 步骤2：添加SecondMe回调
脚本会显示你的IP，例如 `192.168.110.169`

去SecondMe平台添加回调地址：
```
http://192.168.110.169/auth/callback
```

### 步骤3：开始游戏
在局域网内任何设备打开：
```
http://192.168.110.169
```

## 当前检测到的IP
你的局域网IP是：**192.168.110.169**

## 需要添加的SecondMe回调地址
```
http://192.168.110.169/auth/callback
```

## 停止服务
```bash
./stop-lan.sh
```
